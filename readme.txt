Adds paging navigation your WordPress blog.

== Description ==
Conclusion of numbering of pages in the bottom of each page. Very 
convenient in use. There is no necessity of additional options. Changeable 
appearance.

== Installation ==

1. Upload plugin to the `/wp-content/plugins/pagelist/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. If your blog is not located in the root directory, then change the file 'skin/pagelist.css' line 'background: url (/ wp-content/plugins/pagelist/skin/slider.gif) no-repeat 50% 50%'. The same file must be changed if you want to change the appearance of
